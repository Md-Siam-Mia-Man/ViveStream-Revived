# ব্যবহারবিধি (How to Use)

## শুরু করা (Getting Started)

১. **কন্টেন্ট ডাউনলোড বা ইম্পোর্ট করুন**
    - কোনো URL থেকে কন্টেন্ট ডাউনলোড করতে **Downloads** পেজে যান।
    - আপনার কম্পিউটার থেকে লোকাল মিডিয়া যোগ করতে **Settings > Maintenance** এ যান এবং **Import Files** এ ক্লিক করুন।

২. **লাইব্রেরি ম্যানেজ করুন**
    - **Home** এবং **Favorites** পেজে আপনার মিডিয়া দেখা যাবে। ফিল্টার এবং সর্ট বাটন ব্যবহার করে আপনার পছন্দমতো ভিউ সাজিয়ে নিন।
    - **Playlists** এবং **Artists** পেজে গুছানো ভিউ পাবেন। প্লেলিস্ট বা আর্টিস্টে ক্লিক করে তাদের কন্টেন্ট দেখুন এবং কাস্টম কভার ইমেজ আপলোড করুন।
    - লাইব্রেরির যেকোনো কিছু খুঁজে পেতে উপরের সার্চ বারটি ব্যবহার করুন।

৩. **মিডিয়া উপভোগ করুন**
    - প্লেব্যাক শুরু করতে যেকোনো থাম্বনেইলে ক্লিক করুন।
    - "Up Next" কিউ (queue) স্বয়ংক্রিয়ভাবে আপনার প্লেলিস্ট বা আর্টিস্ট অনুযায়ী পরবর্তী ভিডিওগুলো দেখাবে।

## কিবোর্ড শর্টকাট (প্লেয়ার)

| কী (Key)       | অ্যাকশন (Action)       |
| -------------- | ---------------------- |
| `Space` বা `K` | প্লে / পজ (Play / Pause)|
| `M`            | মিউট / আনমিউট (Mute)   |
| `F`            | ফুলস্ক্রিন (Fullscreen) |
| `T`            | থিয়েটার মোড (Theater) |
| `I`            | মিনিপ্লেয়ার (Miniplayer)|
| `←` / `→`      | ৫ সেকেন্ড আগে/পিছে (Seek)|
| `↑` / `↓`      | ভলিউম কমানো/বাড়ানো (Volume)|
| `N`            | পরের মিডিয়া (Next)    |
| `P`            | আগের মিডিয়া (Previous)|
